扩展欧几里得算法(exgcd)¶

注：本文年代久远，谨慎阅读。

Part 0 前置芝士¶

0. 符号与规定¶

  • 对于\(a \bmod b = 0\)，可记作 \(b \mid a\)，其中$ \mid $是整除符号。
  • 对于\(a \bmod c = b \bmod c\)，可记作\(a \equiv b (\bmod\ c)\)，其中$ \equiv $是同余符号。

1. 欧几里得算法（辗转相除法）¶

\[ \gcd(a,b)=\gcd(b,a\bmod b) \]

2. 裴蜀定理¶

对于任意 \(a,b\) （\(a\),\(b\)均为整数）都有 \(x\)，\(y\) 满足 \(ax + by =\gcd\ (a,b)\) 即当且仅当 \(\gcd\ (a,b) \mid c\) 时，\(ax + by = c\) 有解。

3. \(\bmod\)运算的性质¶

  • \((a + b) \bmod p = (a \bmod p + b \bmod p) \bmod p\)
  • \((a - b) \bmod p = (a \bmod p - b \bmod p) \bmod p\)
  • \((a \times b) \bmod p = (a \bmod p \times b \bmod p) \bmod p\)
  • \(a ^ b \bmod p = ((a \bmod p)^b) \bmod p\)

4. 同余方程有解的充要条件¶

对于一个一次同余方程\(ax \equiv b(\bmod\ m)\)，其有解的充分必要条件为\(\gcd(a,m)\mid b\)。 证明

  • 结合律： \(((a+b) \bmod p + c) \bmod p = (a + (b+c) \bmod p) \bmod p\) \(((a\times b) \bmod p \times c)\bmod p = (a \times (b\times c) \bmod p) \bmod p\)

  • 交换律： \((a + b) \bmod p = (b+a) \bmod p\) \((a \times b) \bmod p = (b \times a) \bmod p\)

  • 分配律： \(((a +b)\bmod p \times c) \bmod p = ((a \times c) \bmod p + (b \times c) \bmod p) \bmod p\)

Part 1 定义¶

exgcd求的是$ ax+by=\gcd(a,b) $的一组解， 由裴蜀定理可知，该式一定有解。

Part 2 推导过程¶

拿出纸笔一起来

设有\(x',y'\)满足$ bx'+(a\bmod b)y' = \gcd(b,a \bmod b) $

\(\because a \bmod b=a-b\lfloor \frac{a}{b} \rfloor\)

\(\therefore ay'+b(x'-y'\lfloor \frac{a}{b} \rfloor)=\gcd(b,a \bmod b)\)

\(\because ax+by=\gcd(a,b)\)

又\(\because \gcd(a,b)=\gcd(b,a\bmod b)\)

\(\therefore ax+by=ay'+b(x'-y'\lfloor \frac{a}{b} \rfloor)\)

\(\therefore x=y',y=x'-y'\lfloor \frac{a}{b} \rfloor\)

对于$ bx'+(a\bmod b)y' = \gcd(b,a \bmod b) $，可用相同方法求解

（是不是嗅到了递归的气息？）

递归边界即为\(b=0\)，此时\(x = 1,y = 0\)是\(ax+by=\gcd(a,b)\)的一组解

typedef long long LL;
LL exgcd(LL a,LL b,LL &x, LL &y)
{
    if(b == 0)
    {
        x = 1;
        y = 0;
        return a;
    } 
    LL ans = exgcd(b,a%b,x,y);//递归求x'，y'
    tmp = x;
    x = y;
    y = tmp - a / b * y;
    return ans;
}

那如何求最小非负整数解呢？

\(\because \operatorname{lcm}(a,b)=\frac{ab}{\gcd(a,b)}\)

\(\therefore ax+\operatorname{lcm}(a,b)+by-\operatorname{lcm}(a,b)=\gcd(a,b)\)

变换一下形式可得\(a(x+\frac{b}{\gcd(a,b)})+b(y-\frac{a}{\gcd(a,b)})=\gcd(a,b)\)

显然对于每个\(x+k\frac{b}{\gcd(a,b)}\)，都有一个\(y-k\frac{a}{\gcd(a,b)}\)与之对应

令\(t=k\frac{b}{\gcd(a,b)}\),则\(x+kt\)均为方程的解

所以\((x\bmod t+t)\bmod t\)即为所求

P1082 [NOIP2012 提高组] 同余方程

$ \because ax\equiv 1(\bmod b)$

\(\therefore ax\therefore (ax-1)\bmod b=0\)

\(\therefore ax=kb+1\)

\(\therefore ax-kb=1\)

由于\(k\)是常数，所以

\(ax+bk=1\)

$\because $该式一定有解

\(\therefore\ \gcd(a,b)\mid 1\ \Rightarrow\ \gcd(a,b) = 1\)

\(\therefore\ ax+bk=\gcd(a,b)\)

#include <iostream>
using namespace std;
long long x, y;
void exgcd(int a, int b)
{
    if (b == 0)
    {
        x = 1, y = 0;
        return;
    }
    exgcd(b, a % b);
    long long tx = x;
    x = y;
    y = tx - a / b * y;
}
int main()
{
    int a, b;
    cin >> a >> b;
    exgcd(a, b);
    cout << (x % b + b) % b;//处理负数
}
