T1 parallel¶

前置知识¶

斜率

定义一条直线的斜率 \(k = \dfrac{\Delta y}{\Delta x}\) ，显然，当两条直线斜率相同时，它们不相交就重合。

思考¶

那么我们可以任意选出三个点，计算两两之间的斜率，一定有一个斜率满足要求。

为防止浮点数出奇奇怪怪的锅，请熟练使用比例的基本性质。

首先搞一个 check 函数，用来判断当前斜率是否合法。

bool check(int a, int b)
{
    bool f1 = 0, f2 = 1; //第二条直线，合法标记
    int sec; //在第二条直线上的一个点
    for (int i = 1; i <= n; ++i)
    {
        if (i == a || i == b) continue;
        if (i, a, b在同一条直线上) continue；
        else if (!f1) //还没有第二条直线
            f1 = 1, sec = i; //记录一下
        else
            if (i, sec 这条直线与 a, b 这条直线不平行)
            {
                f2 = 0; break; //不合法，ヾ(￣▽￣)Bye~Bye~
            }
    }
    return f1 && f2; //有第二条直线且合法
}

还需要一个判断三点是否在同一直线上的函数 cmp1(a, b, c)

当直线 \(AB\) 的斜率与直线 \(AC\) 的斜率相等时，由于有公共点 \(A\)，故 \(A, B, C\) 三点共线。

得出判断条件：\(\dfrac{y_a - y_b}{x_a - x_b} = \dfrac{y_a-y_c}{x_a-x_c}\)，稍加化简即为 \((x_a - x_b) \times (y_a - y_c) = (x_a - x_c) \times (y_a - y_b)\)。

还需要一个判断两条直线平行的函数 cmp2(a, b, c, d)

很简单，斜率相等即可：\(\dfrac{y_a-y_b}{x_a-x_b} = \dfrac{y_c-y_d}{x_c-x_d}\)，化简即为 \((x_a-x_b)\times(y_c-y_d)=(x_c-x_d)\times(y_a-y_b)\)

using namespace std;
const int N = 1e5;
int T, n, y[N];

bool cmp1(int a, int b, int c) //同一直线
{
    return (a - b) * (y[a] - y[c]) == (a - c) * (y[a] - y[b]);
}

bool cmp2(int a, int b, int c, int d) //平行
{
    return (a - b) * (y[d] - y[c])  == (d - c) * (y[a] - y[b]);
}

bool check(int a, int b)
{
    bool f1 = 0, f2 = 1; //第二条直线，合法标记
    int sec;
    for (int i = 1; i <= n; ++i)
    {
        if (i == a || i == b || cmp1(i, a, b)) continue;
        else if (!f1)
            f1 = 1, sec = i; 
        else
            if (!cmp2(i, sec, a, b))
            {
                f2 = 0; break;
            }
    }
    return f1 && f2;
}

int main()
{
    scanf("%d", &T);
    while (T--)
    {
        scanf("%d", &n);
        for (int i = 1; i <= n; ++i)
            scanf("%d", &y[i]);
        puts(check(1, 2) || check(1, 3) || check(2, 3) ? "Yes" : "No");
    }
    return 0;
}

思考¶

手玩样例可以发现，答案具有单调性，可以愉快的二分

又可发现，使用 \(d\) 个水管最多可以有 \(k+(k-1-1)+(k-2-1)+\cdots+(k-d)=k\times d + 1 - \sum_{i=1}^di\)

即为 \(k\times d-(1+d)\times d\div 2+1\) 个出口

然后就可以愉快地切掉了

#include <cstdio>
using namespace std;
long long n, k;
int main()
{
    scanf("%lld%lld", &n, &k);
    long long l = 0, r = k, ans;
    while (l <= r) 
    {
        long long mid = (l + r) >> 1;
        long long s = k * mid - (mid + 1) * mid / 2 + 1;
        if (s < n)
            l = mid + 1;
        else
            ans = mid, r = mid - 1;
    } 
    if (k * ans - (ans + 1) * ans / 2 + 1 < n)
        printf("-1\n");
    else
        printf("%lld\n", ans);
    return 0;
}

T3 homework¶

思考¶

该题求最小乘积，很容易就可想到为最小三个数的乘积。

于是排序后便有以下几种情况

  1. \(i,\, j,\, k\)，答案即为 \(k\) 的个数。

  2. \(i,\, j,\, j\)，答案即为 \(C^{2}_{j\text{的个数}}\)

  3. \(i,\, i,\, j\)，答案即为 \(j\) 的个数

  4. \(i,\, i,\, i\)，答案即为 \(C^{3}_{i\text{的个数}}\)

\(O(n)\) 做法：先排序，再计数

高级做法：只用选择排序找出前三小的数，定义 c[i] 表示从 i 开始向后寻找与 a[i] 相等的数的个数，d 表示不相等的数的个数，答案即为 \(\dfrac{c_1 \times c_2 \times c_3} {d!}\)

Warning：\(c_1 \times c_2 \times c_3\) 最大为 \(1e5\times(1e5-1)\times(1e5-2)\)，爆 int 了，所以计算时要乘上 1LL（不然痛失10pts）

#include <bits/stdc++.h>
using namespace std;

int n, a[100010], d = 1, c[5];

int main()
{
    scanf("%lld", &n);
    for (int i = 1; i <= n; ++i)    
        scanf("%lld", &a[i]);
    for (int i = 1; i <= 3; ++i)
    {
        for (int j = i + 1; j <= n; ++j)
            if(a[i] > a[j]) 
                swap(a[i], a[j]);
        for (int j = i; j <= n; ++j)
            if (a[j] == a[i]) c[i]++;
    }
    for (int i = 1; i < 3; ++i)
        if (a[i] == a[i + 1])
            d++;
    long long ans = 1LL * c[1] * c[2] * c[3];
    for (; d; --d)
        ans /= d;
    printf("%lld", ans);
    return 0;
}
